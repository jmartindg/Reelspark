# ReelSpark

Browse and Discover collections of Movies and TV Shows for free.

## Technologies Used

- React
- React Router
- React Query
- React Splide
- React Icons
- TailwindCSS
  - Tailwind Form Plugin
- Axios
- Vite

## Features

- Browse/search movies and tv shows
- Movie and TV Show details
- View similar movies and tv shows
- Browse and view actors and their biography
- View popular movies and tv shows
- View upcoming and now playing movies
- View top rated and airing tv shows
- Movies and tv shows pagination
  - NOTE: To limit API calls from TMDB, page number is limited to 10 only
- Mobile responsive
